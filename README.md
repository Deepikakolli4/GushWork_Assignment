# Gushwork Frontend Assignment

A clean, responsive product showcase built with plain HTML, CSS, and JavaScript.

This project highlights a modern industrial product page with interactive UI patterns: image previews, carousels, FAQ accordion, feature selection states, and in-page lead forms.

## What Makes It Interesting

- Interactive product image gallery with thumbnail zoom preview.
- Multiple carousel experiences (applications, testimonials, logo strip).
- Click-to-highlight feature cards for better content focus.
- In-page modal flows for **Request a Call** and **Book Consultation**.
- Mobile-friendly layout and behavior across sections.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (no framework)

## Run Locally

1. Open the folder in VS Code.
2. Open `index.html` in a browser.
3. For best dev workflow, use Live Server (optional).
